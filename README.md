### PencilKernel
`PencilKernel`被设计为一个跨平台内核,
当前支持平台:`x86` <br/>
将来会支持的平台:`x64` <br/>
制作人员:Lin-2008<br/>
***
### 文件结构
```
doc:相关文档
src:源代码
 |
 +--arch:平台
     |
     +--x86:Intel80x80平台
         |
         +--boot:启动内核相关
         +--fs:文件系统相关
tools:开发工具(windows系统中)

```
### 目标
- [ ] 在文件系统中加载loader和内核
- [ ] 在物理电脑上运行
- [ ] 实现微内核
