### PencilKernel
`PencilKernel`被设计为一个跨平台内核,
当前支持平台:`x86` <br/>
将来会支持的平台:`x64` <br/>
制作人员:Lin Chenjun<br/>
***
### 文件结构
```
doc:相关文档
src:源代码
 |
 +--arch:平台
     |
     +--x86:Intel80x80平台
         |
         +--boot:启动内核相关
         +--
```
### 目标
在物理电脑上运行(未完成)<br/>
实现微内核(未完成)<br/>
